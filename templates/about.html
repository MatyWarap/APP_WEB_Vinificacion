About This Project

This is a Flask-based web application developed to manage the entire winemaking process in a winery. It allows for the digital tracking of grape varieties, winemaking stages (such as fermentation, aging, and bottling), and process parameters (like acidity and pH). The application uses SQLAlchemy for database management and provides a simple, user-friendly interface to register and track grape varieties, as well as monitor the production stages from grape reception to final bottling.

Created by Enzo Rojas, Matías Agüero, Gustavo Vera, Lautaro Videla
